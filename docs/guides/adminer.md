# Adminer - Database management

## Use case

The Adminer template can be used to deploy a database management tool. Adminer is a full-featured database management tool written in PHP. It supports various database systems including MySQL, PostgreSQL, SQLite, and more.

## Template

This guide uses the [adminer](../api-reference/official-templates/databases/adminer.md) template.

## Setup

After [creating the service](./index.md#create-a-service), you can access the Adminer web interface on the port [assigned to your service](./index.md#detailed-view).
